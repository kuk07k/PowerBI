
* Microsoft Power BI
- 많은 데이터에서 무언가를 발견하고, 그것을 통해 효과적인 의사 결정하기 위해 데이터 분석이 필요
- 전문가뿐만 아니라 실무자도 Power BI를 활용하여 손쉽게 데이터를 시각화 및 분석
- 데이터에서 인사이트를 파악하여 효과적인 의사 결정을 할 수 있도록 도와주는 클라우드 데이터 분석 서비스 
- 클라우드 기반의 비즈니스 분석 서비스
- 데이터 세트, 시각화, 보고서, 대시보드, 앱

* Power BI DeskTop
- 데이터 연결
- 연결된 데이터의 정리 및 변환
- 데이터 모델링
- 시각적 개체를 사용한 보고서 만들기
- 보고서를 Power BI 서비스로 게시

* 데이터 모델링 
: 업무 분석 - 주제별로 분류 - 테이블 설계 - 테이블들 간의 관계 작성 
: 구조화된 데이터에서 사용할 연산과 제약 사항 등을 설정하는 과정
